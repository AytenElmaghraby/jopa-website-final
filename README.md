# 🛒 JOPA Dental Store

A modern e-commerce website for dental products built using WordPress, WooCommerce, and a fully custom theme.

---

## 🌐 Live Website

https://jopadental.com/

---

## 📖 Overview

JOPA Dental Store is a responsive e-commerce website developed from scratch by converting a static HTML/CSS design into a custom WordPress theme integrated with WooCommerce.

The project focuses on providing a clean user experience while maintaining a professional online shopping environment.

---

## ✨ Features

- Responsive Design
- Custom WordPress Theme
- WooCommerce Integration
- Product Categories
- Product Variations
- Shopping Cart
- Checkout System
- Contact Page
- About Page
- Custom Product Layout
